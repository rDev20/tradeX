# @tradex/ui

Shared React component library. Built on shadcn/ui + Radix + Tailwind.

## Includes

- shadcn base: Button, Input, Select, Dialog, Drawer, Tabs, Toast, Card, etc.
- Custom tradeX components:
  - `SignalCard` — parsed signal with original-message drawer
  - `ChannelScorecard` — metrics grid + equity sparkline
  - `OrderTimeline` — vertical state-machine viz
  - `FavoriteCard` — expandable to chart overlay
  - `PanicButton` — global, always-visible
  - `ConnectionHealthDot` — broker/Telegram status indicator
  - `KpiTile` — metric with trend arrow
  - `CurrencyInput` — ₹-prefixed, lakh/crore shortcuts
  - `RiskRailBadge` — shows which rail blocked
